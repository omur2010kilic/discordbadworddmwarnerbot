Derin Devlet Botu

Derin Devlet Botu, Discord sunucularında küfür filtresi, otomatik uyarı sistemi ve admin komutları sağlayan bir bottur.

Özellikler

Küfür filtresi: profanity.txt dosyasını oluşturup engellenmesi gereken kelimeleri yazın. Devamında bot halledecek.

DM uyarıları: Kullanıcıya kademeli olarak 3 aşamalı uyarı mesajı gönderir.

Otomatik kick: 4. ihlalde kullanıcıyı sunucudan atar.

Uyarı komutları:

!warnings @kullanıcı → Uyarı sayısını gösterir

!resetwarn @kullanıcı → Admin tarafından uyarıları sıfırlar

Ping komutu: !ping → Botun aktif olduğunu kontrol eder

Restart komutu: Sadece bot sahibi kullanabilir.
